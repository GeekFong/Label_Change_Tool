## **简介**
Label_Change_Tool这个工具主要支持一下几种标注格式的相互转换：

-  yolo的txt格式
-  Pascal VOC的xml格式转换
-  [标注精灵](http://www.jinglingbiaozhu.com/)的xml格式转换（这是我经常使用的标注工具）
